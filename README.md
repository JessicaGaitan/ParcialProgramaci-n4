# Calculador de Consulta Médica

## Situación problemática

En muchas clínicas los pacientes no pueden calcular fácilmente el costo total de su consulta médica dependiendo de la especialidad y exámenes adicionales.

Esta solución está enfocada al sector salud, especialmente clínicas.

---

## Preguntas

### ¿Qué valor agregado tiene el uso de WebComponents?

Permite crear componentes reutilizables, organizados y fáciles de mantener, mejorando la estructura del proyecto.

### ¿De qué forma manipularon los datos sin recargar la página?

Se utilizó el evento submit junto con preventDefault() para evitar la recarga y se modificó el DOM dinámicamente con innerHTML.

### ¿De qué forma validaron las entradas?

Se utilizaron condicionales if para verificar que los campos obligatorios no estuvieran vacíos.

### ¿Cómo manejaría la escalabilidad futura?

Se podrían agregar más especialidades, conectar a base de datos y dividir el proyecto en más componentes reutilizables.
